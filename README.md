# accta

`accta` is a research toolkit for automated coronary artery centreline extraction from contrast-enhanced cardiac CT angiography (CCTA) data, implementing a thirteen-stage pipeline that progresses from raw DICOM ingest through isotropic resampling, lung-mask and blood-pool segmentation, multi-scale Hessian and geometry-based vessel enhancement, ascending aorta isolation and ostia detection, 3-D skeletonisation, gap bridging, anatomical labelling, volume serialisation, and an interactive edit layer with full undo/redo support.
